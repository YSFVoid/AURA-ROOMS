# AURA Rooms

Temporary voice rooms for Discord, powered by discord.js v14.

## Features

- **Join-to-Create** — Join a lobby channel to spawn a private voice room
- **Room Panel** — Rename, set limits, change privacy, kick members, and more via `/room panel`
- **Templates** — Save and apply personal room presets with `/template`
- **Import / Export** — Back up and restore server configuration with `/export` and `/import`
- **Anti-Abuse** — Rate limits, cooldowns, and max-room caps per user
- **Audit Logging** — All actions logged to a configurable text channel

## Commands

| Command | Description |
|---------|-------------|
| `/setup` | One-click server setup (admin) |
| `/room panel` | Open the room control panel |
| `/room activity` | Set room activity tag |
| `/template save\|edit\|delete\|list\|apply` | Manage room templates |
| `/export` | Export server config (admin) |
| `/import` | Import server config (admin) |
| `/about` | Bot info and version |
