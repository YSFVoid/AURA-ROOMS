# AURA Rooms
A production-focused Discord bot for temporary voice room automation (Phase 1).

## Key Features
- Temp Voice room creation from Join-to-Create lobbies
- Room owner control panel (buttons, selects, modals)
- Per-user templates with apply flow
- Config export/import (Replace Exact)
- Anti-abuse protections (cooldown, max rooms, join/leave limiter)
- One-step setup wizard
